## Instalasi / Cara menjalankan di lokal
1. Clone proyek
```bash
  git clone https://github.com/Junia0806/TravelAja-FrontEnd
```
2. Instal library menggunakan npm
```bash
  npm install
```
3. Jalankan proyek
```bash
  npm run dev
```
